# La venganza del potasio

Juego de plataformas 2D hecho en **Godot 4.6.3**.

## Controles

| Tecla | Acción |
|---|---|
| ← → / A D | Moverse |
| ↑ / W / Espacio | Saltar |
| ↓ | atacar o inpulsarte |
| ESC | Pausa |

## Objetivo

Llega hasta el final del nivel esquivando enemigos y recolectando cerezas y gemas para sumar puntos.
